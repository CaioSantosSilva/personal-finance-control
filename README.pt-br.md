# Apex Finance Tracker 💸 (personal-finance-control)

_Leia isto em [Inglês](README.md)_

Um aplicativo moderno, rápido e seguro de controle financeiro pessoal, projetado para ajudar os usuários a monitorar suas receitas, despesas e saúde financeira em tempo real.

---

## 🚀 Tecnologias

Este projeto foi construído utilizando um ecossistema moderno de desenvolvimento full-stack:

*   **Backend:** Python 3.14, FastAPI, SQLAlchemy e SQLite/PostgreSQL.
*   **Frontend:** React, Vite e Tailwind CSS.
*   **Controle de Versão:** Git & GitHub.

---

## 🛠️ Funcionalidades

*   **Autenticação de Usuário:** Sistema seguro de cadastro e login.
*   **Gerenciamento de Transações:** Criação, leitura, atualização e exclusão (CRUD) de receitas e despesas.
*   **Dashboard Financeiro:** Gráficos visuais exibindo o saldo mensal e categorias de gastos.
*   **Filtro por Categorias:** Organização de transações por marcadores personalizados (ex: Alimentação, Aluguel, Salário).
